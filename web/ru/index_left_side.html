UltraDefrag это дефрагментатор дисков с открытым исходным кодом. Может быстро обработать любые файлы, включая те, которые обычно заблокированы Windows, такие как файл подкачки.

НОВОСТИ

UltraDefrag 7.0.0 beta 5 has been released [2015-12-06]

We have finally upgraded wxWidgets to the latest stable 3.0.2 release. It fixes a lot of bugs and provides better debugging facilities which helped us to improve our own code as well. Only drawback is that it uses different language identifiers, so you have to select your language once again. Enjoy!
