UltraDefrag это дефрагментатор дисков с открытым исходным кодом. Может быстро обработать любые файлы, включая те, которые обычно заблокированы Windows, такие как файл подкачки.

НОВОСТИ

UltraDefrag 6.1.0 has been released [2015-04-19]

This release features completely rewritten documentation. A bit later it will be included in UltraDefrag v7 along with a bunch of bug fixes. If you believe some information is missing there, don't hesitate to drop us a note.

 

ИНТЕРФЕЙСЫ

 

ВОЗМОЖНОСТИ

  • Простые но эффективные алгоритмы дефрагментации.
  • Дефрагментация метафайлов и потоков NTFS.
  • Оптимизация таблиц размещения файлов (MFT).
  • Автоматическое отключение компьютера после завершения работы.
  • Ещё...

 

СИСТЕМНЫЕ ТРЕБОВАНИЯ

Настольный ПК: Windows NT 4.0, 2000, XP, Vista, 7, 8

Сервер: Windows NT 4.0, 2000, 2003, 2008, 2008 R2, 2012
