UltraDefrag is an open source disk defragmenter for Windows. It can quickly defragment everything including files which are usually locked by Windows. For instance, UltraDefrag can re-join paging and hibernation files.

NEWS

UltraDefrag 6.1.1 and 7.0.0 beta 3 have been released [2015-11-20]

The first one is a minor maintenance release. It fixes a bug preventing the program to check for available updates reliably. The latter beta release features brand new testing capabilities. Whenever you'll encounter a bug, the program will send us a bit of debugging information automatically. As we'll collect enough information we'll do our best to investigate all cases and release another beta afterwards. Join us at Facebook for more recent news and updates.

FEATURES

  • Simple but efficient defragmentation algorightms.
  • Defragmentation of NTFS metafiles and streams.
  • Optimization of NTFS master file tables (MFT).
  • Automatic hibernation or shutdown after the job completion.
  • More...
