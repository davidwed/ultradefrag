DEVELOPER INFORMATION

This page contains developer information about UltraDefrag.

 

 

 

Contributing to UltraDefrag

If you wish to contribute to UltraDefrag, you need to present your idea to the appropiate person, download the latest source from subversion, make the changes, and submit the patch or improvement. Upon approval, your patch will be committed to subversion and credit added to the file. Person to contact about development are Dmitri Arkhangelski, Justin Dearing.

  • For a full set of development guides refer to the Hanbook's Development Page

 

Who is in Charge

Dmitri is in charge of the main program and the handbook. Justin maintains the documentation and wrote the scheduler. So documentation and scheduler concerns go through Justin, while everything else goes through Dmitri. Sayem maintains the webpage contents.

 

Getting the Source from subversion

The subversion repository url is https://ultradefrag.svn.sourceforge.net/svnroot/ultradefrag. Download the latest copy of the source.

 

Note to Programmers

  • The Build configurator was rewritten in Lua
  • To successfully recompile the driver, simply run configure.pl script and click 'Apply patch to MinGW' button once again

 

Searching through the source

The source is registered with koders. If you are interested in searching the projects code you can either go to the koders project page, or use the search box below.
