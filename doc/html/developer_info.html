Developer Info

This page contains developer information about UltraDefrag.

Contributing to UltraDefrag

If you would like to contribute to UltraDefrag, you need to present you idea to the appropiate person, download the latest source from subversion, make the changes, and submit the patch. Upon approval, your patch will be committed to subversion and credit added to the file.

Who's in Charge

Dmitri is in charge of the main program. Justin maintains the documentation and wrote the scheduler. So documentation and scheduler concerns go through Justin, while everything else goes through Dmitri.

Getting the source from subversion

The subversion repository url is https://ultradefrag.svn.sourceforge.net/svnroot/ultradefrag. Download the latest copy of the source.

Searching through the Source.

The source is registered with koders. If you are interested in searching the projects code you can either to the koders project page, or use the search box below.
