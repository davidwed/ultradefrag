Chapter 2. Installation

To obtain the latest version of the program visit please the project's download page.

There are two different types of binaries available for download:

  1. Full-featured installers. They contain all of the program features, but requires an installation therefore cannot be used directly from the USB sticks.
  2. Portable packages. Download it, unpack it somewhere and use without an installation. Write it on your USB stick and you will be able to use UltraDefrag everywhere by smply running it from there. Note that boot time defragmenter is missing in the portable packages, but the most remaining features are working fine there.

You must have 64-bit version of Windows to run 64-bit versions of the program. IA-64 version was never tested by the authors. It requires Intel Itanium processor and an appropriate 64-bit edition of Windows.

You must be an administrator to install and run UltraDefrag.

You must disable your virus detection software before using UltraDefrag. Any virus scanner running during the defragmentation extremely slows down this process.

It is highly recommended to check Prefer new user mode driver box during an installation. Though old kernel mode driver is a little bit faster, the new driver is much more reliable and also may be used on any latest version of Windows without any additional hacking. The portable version of the program uses this driver, because it requires no system modifications like writing something to registry and copying files to system directories.

If you have strongly decided to use the kernel mode driver, check also this page for the Vista installation guide and the kernel mode driver debugging information.

Portable packages were made especially for people who wants to launch UltraDefrag from USB-flash without an installation. Here is more information about portable applications.

To configure portable package run LanguageSelector.exe tool and select your preferred language.

The UltraDefrag Micro Edition is a special package for advanced users who prefers extreme stability and power. This edition is a little bit faster, more stable (its driver contains just most important parts). It has only console interface to speed up defragmentation process. Boot-time feature is available too. The Micro Edition saves reports in plain text format. You can find them in x:\fraglist.txt file, where x represents letter of the scanned volume.

Note that console interface is very powerful, it has a full set of commands to perform any disk defragmentation job. To show full list of UltraDefrag related commands type ud-help.

You can install UltraDefrag in silent mode by adding /S key to the installer's command in command prompt. In this case no dialogs will be shown and the program will be installed to your hard drive completely with all the features enabled. There is also silent mode in ununstaller program. Type in comand prompt %windir%\UltraDefrag\uninstall.exe /S' to remove the program from your hard drive automatically without any graphical dialogs.
