Chapter 2. Installation

To obtain the latest version of the program visit please the project's download page.

Unfortunately we cannot provide native 64-bit binaries at this moment, because we have no free C compiler which can produce them correctly. To partially resolve this problem we are providing two different types of binaries:

  1. Full-featured installers. They contain all of the program features, but can be used on 32-bit versions of Windows only.
  2. Portable packages. The boot time defragmenter is missing there. On the other hand, you can use the portable packages on any version of Windows.

You must be an administrator to install and run UltraDefrag.

You must disable your virus detection software before using UltraDefrag. Any virus scanner running during the defragmentation extremely slows down this process.

Special notes for Windows Vista users (also applies to any later versions of Windows).

Read this section carefully if you have decided to use kernel mode driver of the program on 32-bit Vista or any later 32-bit version of Windows.

The new versions of Windows are stupid enough: they have few important bugs that reduce your control over your software and even your hardware...

First of all, to install and use UltraDefrag you must disable User Account Control (UAC). This "Windows feature" is one of the most known Vista bugs. Just disable them to fix the problem. You will have no security leaks and mistakes without UAC. Additionally you will be able to install and use many other Open Source programs!

Also you must execute both installer and UltraDefrag GUI only in "Run as administrator" mode. Click right mouse button on the installer icon and choose the appropriate context menu item. Click right mouse button on the UltraDefrag GUI shortcut on your desktop and select properties. Check Run as administrator item.

Finally, you must disable the "Driver signature verification" feature of Windows. This "feature" is one of the most important bugs of the latest Windows versions. Due to them you have no chance to try unsigned drivers, p.a. drivers for your old hardware components! And you cannot try the most powerful Open Source software using it's own drivers. There are at least three ways to resolve this problem:

  • Press F8 button on Windows startup and choose Disable driver signature verification menu item. You should do these steps after each reboot to have an ability to load the UltraDefrag driver. The next solution automates this job.
  • Install the ReadyDriver Plus program. It will automate F8 key pressing on Windows startup and so on.
  • Use the Driver Signature Enforcement Overrider program to digitally sign our driver ultradfg.sys.

Read UltraDefrag Installation Guide for Vista for more details.

You can install UltraDefrag in silent mode by adding /S key to the installer's command in command prompt. In this case no dialogs will be shown and the program will be installed to your hard drive completely with all the features enabled. There is also silent mode in ununstaller program. Type in comand prompt %windir%\UltraDefrag\uninstall.exe /S' to remove the program from your hard drive automatically without any graphical dialogs.
