How to translate the Ultra Defragmenter?

i18n settings of GUI applications are contained in the following files on your system:

  • %windir%\UltraDefrag\ud_i18n.lng - for the main application
  • %windir%\UltraDefrag\ud_config_i18n.lng - for the configurator
  • %windir%\UltraDefrag\ud_scheduler_i18n.lng - for the scheduler

To translate these components and submit your work to our community follow please detailed instructions offered here.
