Translation

How to translate the Ultra Defragmenter?

The translation procedure is very simple and easy. First of all we need gui resources to be translated. To do this important job follow please these steps:

  1. Register on SourceForge.net. After a successful registration you must be logged in to perform the next steps.

  2. Visit the UltraDefrag Wiki Page List.

    • If you wish to improve any existing translation then edit the corresponding document (it's name must be gui.your_language_name, p.a. gui.russian for a russian translation).

    • Otherwise click the New Page link in the left side of the screen. Click Insert Code button and paste into the text area all contents of the gui.us_english file. Than translate everything and click the Save button. Don't forget to add your name and contact information into translated file. The following form is preferred: 
      ;------------------------------------------------------------------------------
; UltraDefrag GUI Andromeda Humans Language Pack.
; Copyright (C) 2008 by Stanley Kubrick (kubrick@andromeda.com)
; Copyright (C) 2010 by Andrey Tarkowski (at@sirius.com)
;------------------------------------------------------------------------------
    • If you have any problems with SF.net Wiki system, read a Wiki help for detailed instructions.

    • Don't use words that are too long, the lengths of the gui controls are fixed! It's easy to check the translated words. Close the UltraDefrag GUI, then replace the contents of %windir%\UltraDefrag\ud_i18n.lng file with your own translation. Run the UltraDefrag GUI again to see how nice it looks.

    • You can replace "Rescan drives" with "Refresh" or "Update" analogs to be short enough. And so on...

  3. If you wish to do anything else then translate the Handbook. Publish your work using UltraDefrag Wiki as described here.

  4. If you have any problems with translation, please send your question to our official Translation Managers Nagy Zsolt and Justin Dearing.

Note

Also Configurator strings should be translated. Instructions are the same as above, but document to be translated is config.us_english.
