Chapter 7. Scheduler

The command line version of UltraDefrag may be run out of system scheduled tasks. The process of adding a scheduled task instance of UltraDefrag is simplified throuth a utility called UltraDefrag Scheduler.NET. This small application requires Microsoft .NET Framework 2.0 to run.

The simple gui has 4 settings:

  1. Whether to run Ultra Defrag every day (Daily) or only certain days of the week (Weekly).
  2. The time of day to run the task.
  3. The drive letter to scan.
  4. Days of the week to defrag if Weekly is selected.
