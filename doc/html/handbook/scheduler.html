Chapter 7. Scheduler

The command line version of UltraDefrag may be run out of system scheduled tasks. The process of adding a scheduled task instance of UltraDefrag is simplified throuth a utility called UltraDefrag Scheduler.

The simple gui has 4 settings:

  1. Whether to run Ultra Defrag every day (Daily) or only certain days of the week (Weekly).
  2. The time of day to run the task.
  3. The drive letter to scan.
  4. Days of the week to defrag if Weekly is selected.

Scheduler can be launched from Start menu. Use Start > Programs > UltraDefrag > Scheduler shortcut. To browse list of all scheduled tasks use the following menu: Start > Settings > Control Panel > Scheduled Tasks.

UltraDefrag Scheduler requires comctl32.dll module version 4.70 or higher to run, therefore you will be able to execute it on Windows 2000 and later systems.

Currently Scheduler is not included in portable version of the program.
