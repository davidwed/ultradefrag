Chapter 10. UltraDefrag Next Generation

UltraDefrag Next Generation is a very simple graphical interface with only three buttons:

The first button opens the disk defragmentation scenario for editing. The second button runs that scenario. The third button closes the window.

UltraDefrag Next Generation highly simplifies the disk defragmentation process. All what you need is to adjust scenario once and than click a single button to perform the complete disk defragmentation.

Insert the hibernate4win now command as described in the console interface section of this document and leave calmly your pc after starting the defragmentation. Your computer will be hibernated automatically when defragmentation job will be completed.

You may specify --small-window command line option when launching the program to run it in compact mode (window size becomes 301x390). When your screen resolution is less than 602x780 this mode is used automatically.
