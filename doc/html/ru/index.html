Последние новости - готов новый стабильный релиз UltraDefrag 4.3!

 

Мы рады вам представить новейший стабильный релиз программы, над которым мы вместе работали последние два месяца. Хотелось бы поблагодарить всех, кто самоотверженно тестировал последние проверочные выпуски и сообщал нам о замеченных ошибках и недостатках. Вместе мы - сила, друзья, мы действительно можем делать высококачественное программное обеспечение. Да здравствует движение Open Source! Да здравствуют патриархи Кен Томпсон, Деннис Ритчи, Брайан Керниган!

Новый релиз программы имеет следующие усовершенствования:

Графический интерфейс:

  • Можно изменять размер окна программы.
  • Окно программы лучше отображается при разных настройках DPI.
  • Размер ячеек кластерной карты можно задавать в опциях программы.
  • Добавлена автоматическая проверка наличия обновления на сервере SourceForge.

Движок:

  • Исправлена ошибка, из-за которой было невозможно дефрагментировать корневой каталог диска.
  • Устранены неточности в отображении MFT, MFT зоны и системных файлов на карте.

Дефрагментация при загрузке Windows:

  • Улучшена индикация прогресса.
  • Введена поддержка backspace и escape клавиш клавиатуры в интерактивном режиме.
  • Исправлено несколько ошибок.

Утилита командной строки:

  • --wait опция добавлена, чтобы облегчить планирование заданий автоматической дефрагментации дисков.
  • Добавлена опция --use-entire-window.

Планировщик:

  • Планировщик больше не входит в состав программы, т.к. стандартный Планировщик Заданий Windows предоставляет гораздо более гибкие возможности для автоматической дефрагментации дисков.

Перевод на другие языки:

  • Обновлены переводы на Чешский, Финский, Японский, Испанский, Португальский, Корейский и Венгерский языки. UltraDefrag это мощная программа для дефрагментации дисков. Первая, среди программ с открытым исходным кодом, которая может обрабатывать файлы, заблокированные системой, такие как файлы реестра и файл подкачки. Программа работает с этими файлами на этапе загрузки Windows, когда они еще не заблокированы.

Другая важнейшая цель программы - максимальная скорость дефрагментации.

UltraDefrag прекрасно работает на Windows NT 4.0, Windows 2000, Windows XP, Windows Server 2003, Windows Vista, Windows Server 2008, Windows 7 и на всех 64-битных версиях Windows.

 

Некоторые интересные возможности UltraDefrag (полный список здесь):

  • Дефрагментация во время загрузки Windows:

Эта функция программы позволяет обрабатывать файлы, которые обычно заблокированы системой, например реестр и файл подкачки.

 

 

  • Выключение компьютера после завершения работы:

Если дефрагментация занимает много времени, включите эту опцию и UltraDefrag выключит ваш компьютер после завершения дефрагментации. Эта опция уменьшает расход электроэнергии и делает ваш компьютер безопаснее.

 

  • Быстрая оптимизация диска:

Оптимизатор перемещает все файлы в начало диска. Это уменьшает время доступа к файлам. Алгоритм намеренно очень прост, так как сложные алгоритмы имеют тенденцию занимать очень много времени.

 

  • Дефрагментация отдельного файла или каталога:

Выберите любой файл или каталог в проводнике Windows и дефрагментируйте его. Эта опция особенно удобна после установки новых игр или софта для ускорения их работы

 

 

  • Простой графический интерфейс:

UltraDefrag имеет простой графический интерфейс,
не отягощенный свистоперделками. Шрифт можно настраивать, общий же стиль окна соответствует выбранной теме Windows.

 

 

  • Планировщик:

Выберите время, когда вам удобно выполнять дефрагментацию, и UltraDefrag сделает вам эту работу точно в заданный срок.
