Whats this page is all about ?

 

Its about keeping track of all the versions of ultradefrag and the changes that has taken place. In many cases you may need to shift back to previous versions if the newer editions are not working for you. Anyway, we have decided to include all versions of UD with download links and the changes that have taken place. The links are from sourceforge.net's UD download archive.
