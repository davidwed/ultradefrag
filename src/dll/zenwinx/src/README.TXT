1. What is ZenWINX?

   ZenWINX is a DLL with useful set of functions for Windows Native applications.
It's a easiest way to write your own native program that will be executed at boot
time in a manner similar to chkdsk and ultradefrag.

   It can be used on Windows NT 4.0, Windows 2000, Windows XP, Windows Server 
2003, Windows Vista and all of the 64-bit editions of Windows.

   More information is available at http://zenwinx.sourceforge.net


2. Features and Limitations

2.1 Features
+ console i/o functions (printf, gets etc.)
+ full x64 support
