ZenWINX

What is ZenWINX?

ZenWinx is a part of the Ultra Defragmenter. It's just a simple framework for windows native applications development. These programs can be executed at boot time in a manner similar to chkdisk. ZenWINX can be used on Windows NT 4.0, Windows 2000, Windows XP, Windows Server 2003, Windows Vista and all of the 64-bit editions of Windows.

ZenWINX comes with ABSOLUTELY NO WARRANTY, for details see the GNU General Public License.

How to build them?

Download the latest sources of the Ultra Defragmenter and follow instructions from INSTALL.TXT file. After a successful build you will have both binaries and headers for your own applications development.
