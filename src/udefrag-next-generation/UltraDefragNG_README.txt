============================================================

To use UltraDefrag Next Generation GUI follow these steps:

1. Download and install the Ultra Defragmenter:

	http://sourceforge.net/projects/ultradefrag/files/

2. Run UltraDefragNG.exe program.

3. Click 'Scenario' button and add all your volumes there.

4. Click 'Start' button.
